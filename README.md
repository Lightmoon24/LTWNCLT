# 🚀 Jilow - Website Quản Lý Công Việc

> **Jilow** là hệ thống quản lý công việc được lấy cảm hứng từ **Trello** và **Jira**, giúp người dùng dễ dàng theo dõi tiến độ, quản lý nhiệm vụ và trao đổi trong nhóm.

# 🎯 Mục Tiêu

Xây dựng một website hỗ trợ quản lý công việc theo mô hình Kanban tương tự **Trello** kết hợp các tính năng quản lý dự án của **Jira**, giúp:

- 📋 Quản lý công việc
- 👥 Quản lý thành viên
- 💬 Trao đổi trực tiếp
- 📊 Theo dõi tiến độ dự án
